# 🌌 AstraNova Zenith – Precision Astrology Suite

AstraNova Zenith is an interactive, front-end based web application that provides personalized astrology insights using Western Astrology, Vedic Astrology, and Numerology. It delivers real-time results with a modern, visually appealing user interface.

---

## ✨ Features

- **Western Astrology**
  - Sun Sign detection
  - Zodiac personality description
  - Interactive zodiac wheel

- **Vedic Astrology**
  - Lagna (Rising Sign) calculation
  - D1 Birth Chart visualization
  - Mahadasha prediction
  - Remedies (gemstone, mantra, auspicious day)

- **Numerology**
  - Life Path Number calculation
  - Cosmic number insights

- **Compatibility Check**
  - Match score based on zodiac elements
  - Relationship verdict

- **Tarot Card Feature**
  - Random tarot card generation
  - Card meaning and guidance

- **Oracle Predictions**
  - Instant answers to user questions

- **Journal Module**
  - Save personal notes with timestamps

- **Real-Time Processing**
  - No page reload required

- **Modern UI Design**
  - Glassmorphism effects
  - Neon styling and animations
  - Responsive layout

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- SVG (for charts)  
- Google Fonts  

---
## 📌 How It Works

1. User enters birth details (date and time)  
2. System validates the input  
3. JavaScript processes astrology calculations  
4. Results are displayed dynamically on the UI  
5. Additional modules provide extra insights (compatibility, tarot, oracle, journal)
