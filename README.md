🌌 AstraNova Zenith – Precision Astrology Suite

AstraNova Zenith is a modern, interactive web-based astrology application that provides personalized astrological insights using Western Astrology, Vedic Astrology, and Numerology. The application is fully client-side and delivers real-time results with a visually immersive UI.

✨ Features
🔮 Western Astrology
Sun Sign detection
Zodiac personality insights
Interactive zodiac wheel
🕉️ Vedic Astrology
Lagna (Rising Sign) calculation
D1 Birth Chart visualization
Mahadasha prediction
Suggested remedies (gemstone, mantra, day)
🔢 Numerology
Life Path Number calculation
Cosmic number insights
❤️ Compatibility Check
Match score based on zodiac elements
Relationship verdict
🃏 Tarot Card Reader
Random tarot card generator
Meaning and guidance
🔮 Oracle Predictions
Instant answers to user questions
📓 Personal Journal
Save reflections with timestamps
⚡ Real-Time Processing
Instant results without page reload
🎨 Modern UI
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & animations
JavaScript (Vanilla JS) – Logic & interactivity
📌 How It Works
User enters birth details (date & time)
System validates input
JavaScript processes astrology calculations
Results are displayed dynamically
Additional modules provide insights (match, tarot, oracle, journal)
